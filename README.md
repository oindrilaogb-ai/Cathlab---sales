# Cathlab Sales & Marketing Strategy

## 📌 Project Overview
This project presents a sales and marketing strategy for Siemens Healthineers Cath Lab systems. The presentation analyzes customer behavior, competitor positioning, and strategies to increase market share.

## 🎯 Objectives
- Understand the Cath Lab buying journey
- Analyze customer decision-making
- Study competitors like Philips and GE
- Recommend sales and marketing strategies

## 🛠️ Tools Used
- Microsoft PowerPoint
- Microsoft Excel
- Market Research
